Retrograde is the simplest Bach Math operation: play the notes of a theme in reverse order. The last note becomes first. The first becomes last. The rhythm reverses too — if the original is long-short-short, the retrograde is short-short-long.

## What Retrograde Sounds Like

A retrograded theme sounds like the same character moving in the opposite direction through time. The intervals are all familiar — the listener has heard every one of them in the original — but the direction of travel has reversed. It sounds like viewing a journey from its destination looking backward, rather than from the origin looking forward.

This is different from simply feeling "backwards." Bach's retrograde subjects sound strange and familiar simultaneously. The ear recognises the pitch material but cannot immediately place it. Recognition arrives slightly delayed. That slight delay is compositionally useful — it creates a flicker of uncertainty before resolution.

In Contrapunctus VII of the *Art of Fugue*, the subject appears simultaneously in original and retrograde form. One voice is arriving as another is departing. The effect is not confusion but depth — the same musical thought is present at multiple points in its own timeline at once.

## What Retrograde Means Compositionally

Retrograde is most naturally used when a theme needs to represent:

- **Return to origin** — a character who has made a complete circuit and now moves back toward where they started, but from the other direction
- **Temporal reversal** — a character whose relationship to cause and effect has changed; things that used to be results now feel like causes
- **Structural closure** — the end of a work that is literally moving toward its own beginning
- **A character seen from behind** — the same identity, but the perspective has shifted from "going" to "arriving"

## In Sonic Pi

```ruby
# Using the theme from Lesson 1
theme = [
  [:e4, 0.5], [:b4, 1.0], [:g4, 0.25], [:g4, 0.25],
  [:a4, 0.5], [:f#4, 1.5]
]

# Retrograde: reverse the array of [pitch, duration] pairs
theme_retro = theme.reverse
# → [[:f#4, 1.5], [:a4, 0.5], [:g4, 0.25], [:g4, 0.25], [:b4, 1.0], [:e4, 0.5]]

define :play_theme do |t, amp: 0.7|
  t.each do |p, d|
    play p, sustain: d * 0.85, amp: amp
    sleep d
  end
end

# Hear original and retrograde in sequence
live_loop :comparison do
  play_theme theme
  sleep 1
  play_theme theme_retro, amp: 0.55
  sleep 1
end
```

## Pitch-Only Retrograde vs. Full Retrograde

You can retrograde only the pitch sequence while keeping the original rhythm:

```ruby
pitches   = theme.map { |p, _| p }
durations = theme.map { |_, d| d }

retro_pitches = pitches.reverse
# Durations stay the same — same rhythmic feel, reversed pitch contour

retro_pitches.zip(durations).each do |p, d|
  play p, sustain: d * 0.85
  sleep d
end
```

Or retrograde only the rhythm while keeping the original pitches:

```ruby
retro_durs = durations.reverse
pitches.zip(retro_durs).each do |p, d|
  play p, sustain: d * 0.85
  sleep d
end
```

Each variant has a different character. Full retrograde reverses everything. Pitch-only retrograde keeps the rhythmic identity while inverting the melodic direction. Duration-only retrograde changes the time feel while preserving the pitch contour. All three are useful in different contexts.

## Hearing the Relationship

```ruby
# Play all four variants in sequence to hear the family resemblance
live_loop :retrograde_study do
  puts "Original"
  play_theme theme

  sleep 0.5

  puts "Full retrograde"
  play_theme theme.reverse

  sleep 0.5

  puts "Pitch retrograde, original rhythm"
  pitches   = theme.map { |p, _| p }
  durations = theme.map { |_, d| d }
  pitches.reverse.zip(durations).each { |p, d| play p, sustain: d*0.85; sleep d }

  sleep 0.5

  puts "Original pitch, duration retrograde"
  pitches.zip(durations.reverse).each { |p, d| play p, sustain: d*0.85; sleep d }

  sleep 2
end
```

*Project connection: In a four-act programmatic work, retrograde most naturally appears in Act IV, where material returns toward its origin. A character whose Act IV theme is the retrograde of their Act I theme has made a complete arc — not repetition, but return transformed by the journey.*
