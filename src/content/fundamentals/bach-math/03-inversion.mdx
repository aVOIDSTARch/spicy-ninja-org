Inversion flips every interval in a theme: a rising step becomes a falling step, a leap upward becomes a leap downward. If the original moves +7 then -4 then -3, the inversion moves -7 then +4 then +3. The theme is reflected around a central axis pitch — everything that went up now goes down.

## What Inversion Sounds Like

An inverted theme sounds like the same character with a reversed emotional direction. Rising intervals convey aspiration, arrival, reaching. Falling intervals convey descent, acceptance, return. Inversion doesn't change which intervals are present — it changes their direction.

A theme that originally rises — forward-looking, searching — inverted becomes a theme that falls. Same interval sizes, same rhythm, same note-to-note distances, but the emotional trajectory reversed. It sounds like the original character seen from the other side of their relationship with someone.

This is what makes inversion more musically interesting than retrograde for most melodic material: where retrograde reverses the *order* of events, inversion reverses the *direction* of each event. The result is simultaneously familiar and tonally altered.

## Why Bach Uses Inversion

In fugue, inversion creates the countersubject — the melodic companion that runs against the main subject. Because its intervals are mirror images of the subject's, the two voices tend to produce stable intervals (thirds, sixths) when combined, rather than collisions. The mathematics of the mirror relationship produces natural harmonic compatibility.

This is not accident. Bach designed his subjects specifically so that the inversion would be harmonically compatible with the original. In the *Art of Fugue*, subjects and inversions are combined simultaneously throughout — two voices at once, one the mirror of the other.

## In Sonic Pi: Computing Inversion

```ruby
theme = [
  [:e4, 0.5], [:b4, 1.0], [:g4, 0.25], [:g4, 0.25],
  [:a4, 0.5], [:f#4, 1.5]
]

pitches   = theme.map { |p, _| p }
durations = theme.map { |_, d| d }

# Convert to MIDI integers for arithmetic
midi = pitches.map { |p| note(p) }
# → [64, 71, 67, 67, 69, 66]

# Inversion: negate each interval from the first note
root = midi.first
inverted_midi = midi.map { |m| root - (m - root) }
# → [64, 57, 61, 61, 59, 62]
# E4, A3, Db4, Db4, B3, D4

inverted_pitches = inverted_midi.map { |m| m.to_sym rescue m }

define :play_theme do |p_arr, d_arr, amp: 0.7|
  p_arr.zip(d_arr).each do |p, d|
    play p, sustain: d * 0.85, amp: amp
    sleep d
  end
end

live_loop :inversion_study do
  puts "Original"
  play_theme pitches, durations
  sleep 0.5
  puts "Inversion"
  play_theme inverted_midi, durations, amp: 0.55
  sleep 1
end
```

## The Four Combinatorial Forms

Any theme now has four forms available:

| Form | Operation | Description |
|---|---|---|
| **O** (Original) | — | As written |
| **R** (Retrograde) | Reverse the array | Played backwards |
| **I** (Inversion) | Negate intervals from root | Upside down |
| **RI** (Retrograde Inversion) | Invert, then reverse | Backwards and upside down |

```ruby
# Generate all four forms
original         = pitches.zip(durations)
retrograde       = original.reverse
inversion        = inverted_midi.zip(durations)
retro_inversion  = inversion.reverse

# Play all four
[
  ["O",  pitches,         durations],
  ["R",  pitches.reverse, durations.reverse],
  ["I",  inverted_midi,   durations],
  ["RI", inverted_midi.reverse, durations.reverse]
].each do |label, ps, ds|
  puts label
  play_theme ps, ds
  sleep 0.5
end
```

Retrograde inversion sounds like the original heard from very far away, or very long after — still related, but transformed by maximum distance.

## Combining O and I Simultaneously (Counterpoint)

The payoff of inversion is running O and I simultaneously:

```ruby
live_loop :voice_o do
  play_theme pitches, durations, amp: 0.7
end

live_loop :voice_i do
  play_theme inverted_midi, durations, amp: 0.6
end
```

Listen to the harmonies at each intersection. They tend toward thirds and sixths because the intervals are mirror images — the very property that makes them sound related also makes them stack stably. This is Bach's contrapuntal engine in its simplest form.
